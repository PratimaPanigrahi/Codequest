Python Standard Library

Powerful tools built into Python

What is the Standard Library?

The Python Standard Library is a collection of modules that come with every Python installation. These modules provide a wide range of functionality.

Key benefits:

  • No need for external installations
  • Well-tested and reliable code
  • Comprehensive documentation
  • Cross-platform compatibility
Did you know? Python's "batteries included" philosophy means it comes with many useful modules right out of the box.
# Importing modules
import math
import random
import datetime

# Using module functions
root = math.sqrt(25)
num = random.randint(1, 10)
now = datetime.datetime.now()

print("Results:", root, num, now)

The Random Module

Generate random numbers for unpredictability

Random Number Generation

The random module provides functions for generating random numbers.

Key features:

  • Generate random integers and floats
  • Randomly select elements from sequences
  • Shuffle sequences in place
  • Generate random samples
Random Fact: Python's random module uses a pseudo-random number generator.
# Import the random module
import random

# Generate random numbers
random_int = random.randint(1, 100)
random_float = random.random()
random_range = random.uniform(1.5, 5.5)

# Working with sequences
colors = ["red", "blue", "green"]
random_color = random.choice(colors)
random.shuffle(colors)

print("Random integer:", random_int)
print("Random color:", random_color)

Random Module Applications

Practical uses of randomness

Real-World Applications

The random module is useful in many scenarios:

Games

Dice rolls, card shuffling, random events

Simulations

Monte Carlo methods, statistical modeling

Security

Password generation, cryptographic nonces

Security Note: For cryptographic purposes, use the secrets module instead of random.
# Example: Password generator
import random
import string

def generate_password(length=8):
  chars = string.ascii_letters + string.digits
  password = ''.join(random.choice(chars) for _ in range(length))
  return password

# Example: Lottery number picker
def lottery_numbers(count=6, max_num=49):
  numbers = list(range(1, max_num+1))
  return random.sample(numbers, count)

print("Password:", generate_password(10))
print("Lottery:", lottery_numbers())

The Datetime Module

Working with dates and times

Date and Time Manipulation

The datetime module provides classes for manipulating dates and times.

Key classes:

  • datetime - combines date and time
  • date - represents a date
  • time - represents a time
  • timedelta - represents a duration
Time Fact: Python's datetime can handle dates from year 1 to 9999.
# Import the datetime module
import datetime

# Get current date and time
now = datetime.datetime.now()
today = datetime.date.today()

# Create specific dates
birthday = datetime.date(1990, 5, 15)
meeting = datetime.datetime(2023, 12, 25, 18, 0)

# Calculate differences
age = today - birthday
future = now + datetime.timedelta(days=7)

print("Now:", now)
print("Today:", today)
print("Next week:", future)

Datetime Formatting

Displaying dates in different formats

Formatting Dates

The strftime() method allows you to format datetime objects as strings.

Common format codes:

  • %Y - Year with century (2023)
  • %m - Month as number (01-12)
  • %d - Day of month (01-31)
  • %H - Hour (00-23)
  • %M - Minute (00-59)
  • %A - Full weekday name
  • %B - Full month name
Tip: Use strptime() to convert strings to datetime objects.
# Date formatting examples
import datetime

now = datetime.datetime.now()

# Different date formats
format1 = now.strftime("%Y-%m-%d")
format2 = now.strftime("%B %d, %Y")
format3 = now.strftime("%A, %B %d at %I:%M %p")

# Parsing dates from strings
date_str = "2023-12-25"
parsed_date = datetime.datetime.strptime(date_str, "%Y-%m-%d")

print("Format 1:", format1)
print("Format 2:", format2)
print("Format 3:", format3)
print("Parsed:", parsed_date)

Combining Modules

Using multiple modules together

Module Integration

These standard library modules can be combined to create powerful applications.

Example scenarios:

  • Game with random events and timing
  • Data analysis with mathematical functions
  • Scheduling system with random assignments
  • Simulation with mathematical models
Integration Tip: Python's modules are designed to work well together.
# Example: Event scheduler
import datetime
import random

events = ["Meeting", "Lunch", "Training"]
today = datetime.date.today()
schedule = []

for i in range(5):
  day = today + datetime.timedelta(days=i)
  event = random.choice(events)
  time = datetime.time(random.randint(9, 17), 0)
  schedule.append((day, time, event))

for day, time, event in schedule:
  print(f"{day.strftime('%A')} at {time}: {event}")
