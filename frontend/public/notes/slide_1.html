What is Python?

A high-level, interpreted programming language

Python is a powerful, versatile programming language known for its simple syntax and readability. It emphasizes code readability with its clean, English-like syntax.

Python supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

  • High-level - Easy for humans to read and write
  • Interpreted - Code is executed line by line
  • Dynamically typed - No need to declare variable types
  • Open source - Free to use and modify
Fun Fact: Python was named after Monty Python, not the snake!
# Simple Python code example
def greet(name):
    return f"Hello, {name}!"

# Using the function
message = greet("Python Learner")
print(message)

Creator & History

The story behind Python's creation

Python was created by Guido van Rossum in the late 1980s. He started implementing Python in December 1989.

Van Rossum named the language after the British comedy group Monty Python, not the snake. He wanted a name that was short, unique, and slightly mysterious.

"Python is an experiment in how much freedom programmers need. Too much freedom and nobody can read another's code; too little and expressiveness is endangered."
- Guido van Rossum

Python Syntax Overview

Understanding Python code structure

Python syntax is designed to be readable and straightforward. Key characteristics include:

  • Indentation - Uses whitespace to define code blocks (no curly braces)
  • Simple statements - One statement per line (generally)
  • Dynamic typing - No need to declare variable types
  • Clear naming - Descriptive function and variable names
# Simple Python example
name = "Alice"
age = 30

if age >= 18:
    print(f"Hello {name}, you are an adult")
else:
    print(f"Hello {name}, you are a minor")

Comments in Python

Adding explanations to your code

Comments are essential for explaining code and making it more readable. Python supports two types of comments:

  • Single-line comments - Start with # and continue to the end of the line
  • Multi-line comments - Using triple quotes (''' or """) for documentation

Good comments help you and others understand the purpose of your code.

# This is a single-line comment
name = "Bob" # This comment is after code

'''
This is a multi-line comment
It can span several lines
Often used for documentation
'''

# Calculate area of a circle
radius = 5
area = 3.14159 * radius ** 2
print(f"Area: {area}")

Print Function

Displaying output in Python

The print() function is used to display output to the console. It's one of the most commonly used functions in Python.

Key features of the print function:

  • Can print strings, numbers, variables, and expressions
  • Accepts multiple arguments separated by commas
  • Automatically adds a space between arguments
  • Includes optional parameters like sep and end
# Basic print examples
print("Hello, World!")
print(42)
print("Python", "is", "awesome")

# Printing variables
name = "Alice"
age = 25
print("Name:", name, "Age:", age)

# Using f-strings (formatted strings)
print(f"Hello, {name}! You are {age} years old.")
