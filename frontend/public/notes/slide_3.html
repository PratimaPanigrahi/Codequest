Control Flow: Conditionals

Making decisions in your Python programs

Control flow allows your program to make decisions and execute different code blocks based on conditions. Conditionals are the foundation of decision-making in programming.

In this lesson, you'll learn:

  • How to use the if statement for single conditions
  • Using if/else for two possible outcomes
  • Handling multiple conditions with elif
  • Nesting conditional statements for complex logic
Did you know? Conditional statements are also called "branching" because they create different paths (branches) for your code to follow.
# Simple conditional example
temperature = 25

if temperature > 30:
    print("It's a hot day!")
elif temperature > 20:
    print("It's a pleasant day.")
else:
    print("It's a bit chilly.")

The if Statement

Make a decision based on a single condition

The if statement is the simplest form of conditional. It executes a block of code only if a condition is True.

Syntax:

if condition:
    # code to execute if condition is True

Key points:

  • The condition must evaluate to True or False
  • Code inside the if block is indented (usually 4 spaces)
  • If the condition is False, the code block is skipped
# Example 1: Simple if statement
age = 18

if age >= 18:
    print("You are eligible to vote.")

# Example 2: Using comparison operators
score = 85

if score > 80:
    print("Great job! You passed with distinction.")

# Example 3: Checking if a value exists
name = "Alice"

if name:
    print(f"Hello, {name}!")

if/else Logic

Handle two possible outcomes based on a condition

The if/else statement allows you to specify two different code blocks: one to execute if the condition is True, and another if it's False.

Syntax:

if condition:
    # code if condition is True
else:
    # code if condition is False

Key points:

  • Exactly one of the two code blocks will execute
  • The else clause doesn't have a condition
  • This is useful for binary decisions (yes/no, true/false)
# Example 1: Simple if/else
temperature = 25

if temperature > 30:
    print("It's hot outside.")
else:
    print("It's not too hot.")

# Example 2: Checking user access
is_logged_in = True

if is_logged_in:
    print("Welcome to your account!")
else:
    print("Please log in to continue.")

# Example 3: Number parity check
number = 7

if number % 2 == 0:
    print(f"{number} is even.")
else:
    print(f"{number} is odd.")

elif for Multiple Conditions

Handle multiple conditions and outcomes

The elif (else if) statement allows you to check multiple conditions in sequence. It's used after an if statement and before an optional else.

Syntax:

if condition1:
    # code if condition1 is True
elif condition2:
    # code if condition2 is True
else:
    # code if all conditions are False

Key points:

  • Conditions are checked in order from top to bottom
  • Only the first True condition's code block executes
  • You can have multiple elif statements
  • The else clause is optional
# Example 1: Grade classification
score = 85

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
elif score >= 60:
    print("Grade: D")
else:
    print("Grade: F")

# Example 2: Temperature ranges
temp = 22

if temp > 30:
    print("Hot")
elif temp > 20:
    print("Warm")
elif temp > 10:
    print("Cool")
else:
    print("Cold")

Comparison Operators

Tools for creating conditions

Comparison operators are used to compare values and return True or False. They are essential for creating conditions in conditional statements.

Operator Description Example
== Equal to 5 == 5 → True
!= Not equal to 5 != 3 → True
> Greater than 5 > 3 → True
< Less than 5 < 3 → False
>= Greater than or equal to 5 >= 5 → True
<= Less than or equal to 5 <= 3 → False
Remember: The equality operator is == (double equals), not = (single equals) which is for assignment.
# Examples of comparison operators
x = 10
y = 5

# Equality check
if x == 10:
    print("x is 10")

# Inequality check
if x != y:
    print("x and y are different")

# Greater than
if x > y:
    print("x is greater than y")

# Less than or equal to
if y <= 5:
    print("y is less than or equal to 5")

# Combining conditions
if x > 5 and y < 10:
    print("Both conditions are True")

Logical Operators

Combining multiple conditions

Logical operators allow you to combine multiple conditions in a single statement.

Operator Description Example
and True if both conditions are True (5 > 3) and (5 < 10) → True
or True if at least one condition is True (5 > 3) or (5 < 2) → True
not Reverses the result (True becomes False) not(5 > 3) → False

Operator precedence (from highest to lowest):

  1. not
  2. and
  3. or
Tip: Use parentheses to make complex conditions clearer and to control evaluation order.
# Examples of logical operators
age = 25
has_license = True

# AND operator - both conditions must be True
if age >= 18 and has_license:
    print("You can drive a car.")

# OR operator - at least one condition must be True
is_weekend = False
is_holiday = True

if is_weekend or is_holiday:
    print("It's a day off!")

# NOT operator - reverses the condition
is_raining = False

if not is_raining:
    print("You don't need an umbrella.")

# Combining multiple operators
temperature = 22
is_sunny = True

if (temperature > 20 and temperature < 30) and is_sunny:
    print("Perfect weather for a walk!")

Best Practices

Writing clean and efficient conditional code

Follow these best practices to write clean, readable, and efficient conditional statements:

  • Keep conditions simple: Break complex conditions into smaller, named variables
  • Use meaningful variable names: Makes conditions easier to understand
  • Avoid deep nesting: Consider using early returns or separate functions
  • Order conditions logically: Place most likely or important conditions first
  • Use parentheses for clarity: Especially with complex logical expressions
Pythonic Tip: Python supports conditional expressions (ternary operator) for simple if/else assignments: x = value1 if condition else value2
# GOOD: Clear, readable conditions
age = 25
has_license = True
has_car = False

# Use descriptive variable names for conditions
can_drive = age >= 18 and has_license
needs_rental = can_drive and not has_car

if can_drive:
    if needs_rental:
        print("You can drive but need to rent a car.")
    else:
        print("You're all set to drive!")
else:
    print("You cannot drive.")

# Using ternary operator for simple assignments
temperature = 25
weather_message = "Hot" if temperature > 30 else "Comfortable"
print(weather_message)
