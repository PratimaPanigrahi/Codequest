Guess the Number Game

Using loops and conditionals

The "Guess the Number" game is a classic that teaches:

  • Random number generation
  • While loops for repeated attempts
  • Conditional statements for feedback
  • User input validation

Game Flow

1. Computer selects a random number
2. Player guesses the number
3. Computer provides hints (too high/too low)
4. Repeat until correct guess

# Guess the Number implementation
import random

def guess_number():
    number = random.randint(1, 100)
    attempts = 0

    print("I'm thinking of a number between 1 and 100.")

    while True:
        guess = int(input("Your guess: "))
        attempts += 1

        if guess < number:
            print("Too low!")
        elif guess > number:
            print("Too high!")
        else:
            print(f"Correct! You guessed it in {attempts} attempts.")
            break

Quiz Application

Using data structures for questions and answers

A quiz app demonstrates how to:

  • Store data in lists and dictionaries
  • Iterate through questions
  • Track and display scores
  • Handle different question types

Quiz Structure

1. Define questions and answers
2. Present questions one by one
3. Check answers and update score
4. Display final results

# Simple Quiz Implementation
questions = [
    {
        "question": "What is the capital of France?",
        "answer": "Paris"
    },
    {
        "question": "Which planet is known as the Red Planet?",
        "answer": "Mars"
    }
]

score = 0

for q in questions:
    print(q["question"])
    user_answer = input("Your answer: ")
    if user_answer.lower() == q["answer"].lower():
        print("Correct!")
        score += 1
    else:
        print(f"Wrong! The answer was {q['answer']}")

print(f"Your final score: {score}/{len(questions)}")

Choose Your Own Adventure

Using nested conditional statements

These games create branching narratives where player choices determine the story's direction.

Key concepts:

  • Nested if-elif-else statements
  • Story branching and multiple endings
  • Player choice impact on narrative
  • Game state tracking
Fun Fact: The "Choose Your Own Adventure" book series sold over 250 million copies worldwide.
# Simple Adventure Game Structure
def start_adventure():
    print("You're in a forest. Do you go LEFT or RIGHT?")
    choice1 = input("> ").lower()

    if choice1 == "left":
        print("You find a river. SWIM across or FOLLOW the river?")
        choice2 = input("> ").lower()
        if choice2 == "swim":
            print("You safely cross and find treasure! You win!")
        else:
            print("You get lost in the forest. Game over.")
    else:
        print("You encounter a bear! RUN or HIDE?")
        choice2 = input("> ").lower()
        if choice2 == "run":
            print("You escape safely! You win!")
        else:
            print("The bear finds you. Game over.")

# Input validation example
def get_valid_input(prompt, valid_options):
    while True:
        user_input = input(prompt).lower()
        if user_input in valid_options:
            return user_input
        else:
            print(f"Please enter one of: {', '.join(valid_options)}")

Debugging and Testing Games

Finding and fixing issues in your code

Effective debugging strategies for game development:

  • Print debugging: Use print statements to track values
  • Step-through execution: Trace through your code line by line
  • Test edge cases: Check boundary conditions
  • User testing: Have others play your game
  • Error handling: Anticipate and manage potential errors

Common Game Bugs

• Infinite loops
• Incorrect condition checks
• Off-by-one errors
• Input handling issues
• Logic flow problems

# Debugging example with print statements
def debug_game():
    secret_number = 42
    attempts = 0
    max_attempts = 5

    print(f"DEBUG: Secret number is {secret_number}")

    while attempts < max_attempts:
        print(f"DEBUG: Attempt {attempts + 1} of {max_attempts}")
        guess = int(input("Guess: "))
        print(f"DEBUG: User guessed {guess}")

        if guess == secret_number:
            print("Correct!")
            break
        elif guess < secret_number:
            print("Too low")
        else:
            print("Too high")

        attempts += 1
        print(f"DEBUG: Attempts counter: {attempts}")
